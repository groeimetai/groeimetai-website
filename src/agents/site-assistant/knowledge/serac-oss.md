# Serac — onze open source toolkit

**Repo:** github.com/serac-labs/serac
**Licentie:** Elastic v2
**Status:** v1.x in productie

## Wat het is

Serac is een agent-framework voor ServiceNow flows. Orchestreert tool calls, audit log en menselijke approval-stap waar het ertoe doet. Geschreven in TypeScript.

Begonnen als klantwerk. Open source gemaakt omdat we vinden dat klanten geen vendor lock-in mogen krijgen op iets wat hun eigen processen automatiseert.

## Waarom dit relevant is voor de site-assistent

Architecten gebruiken Serac in productie zonder dat we ze ooit gesproken hebben. Dat is de proof, niet een testimonial-pagina:
